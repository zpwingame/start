交互动效:
1.默认进来:展现是在一个动态的知识星空中，具体体现背景星星闪烁，一会大、一会小，再消失; 背景星球缓慢旋转或者上下摆动;学习中星球，上下摆动。
S 曲线是一个渐变的过程，已完成星球S 曲线是淡蓝色发光状态，待学习/学习中星球S 曲线是蓝绿色;待解锁星球及后面的曲线是浅蓝色。
2.页面滑动
- 上下滑动 :上下滑动页面查看下面已完成和上面待解锁/待学习的任务星球- 上滑 : 查看已完成的任务星球，已完成的任务星球沿着 S 形曲线向上滑动,- 下滑 : 查看未解锁/待学习的任务星球，未解锁/待学习的星球沿着S 形曲线向下滑动- 动画效果 : 滑动时，背景不动，星球固定在 S 形曲线，曲线和星球整体缓缓滑动。
3.星球点击
- 点击待解锁星球 : 星球轻微晃动(左右摇摆)，弹出“请先完成全部必修课程，才能解锁考试哦”弹框。
- 点击待学习/学习中/待考试星球 :星球短暂高亮 (发光效果) ，星球放大 (1.2 倍)，进入课程详情/考试详情页面，
- 点击已完成星球 : 星球轻微放大 (1.1 倍)，进入课程详情/考试详情页面
